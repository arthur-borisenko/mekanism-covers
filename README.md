<h3 align="center">Mekanism Covers</h3>
<p align="center">A Mekanism addon that adds a simple way to cover its cables and pipes(backported to 1.16.5).</p>

## Features
#### How to use

Craft the cover and combine it with the desired block in the crafting table. After that, just right-click the cable or pipe you want to cover.

If you want to remove the cover, just break the block, use a Configurator in the wrench mode, or a wrench of another tech mod.

#### Crashing with Iris, Oculus, Sodium, Embeddium, etc or rendering behaves strangely?

If you're having any crashes or unexpected behavior while using mods that change Minecraft's rendering. Go into the config file located at your .minecraft/config/mekanismcovers.json, and set disableAdvancedCoverRendering to true.
Some features may be untested while backporting but it should work fine.
